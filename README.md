# FASTA Sequence Analyzer

This project analyzes biological sequences stored in FASTA format.
It calculates sequence length and GC content for each sequence and
exports the results to a CSV file.

## Features
- Reads FASTA files
- Counts total sequences
- Calculates sequence length
- Calculates GC content
- Outputs results to CSV

## Requirements
- Python 3.8+
- Biopython

## How to Run
```bash
pip install -r requirements.txt
python analyzer.py
